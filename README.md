# GAME-Homebound
# 📖 Resumo / Summary / Resumen:

🇧🇷 Português
Homebound é um jogo de plataforma 2D desenvolvido em Python (Pygame). Diferente dos jogos tradicionais de heróis, aqui o protagonista é um trabalhador exausto. O desafio não é derrotar monstros, mas sim desviar das pressões sociais (o chefe, cobranças familiares e distrações) para finalmente chegar em casa, largar a bolsa e encontrar o seu único refúgio: o abraço da esposa.

🇺🇸 English
Homebound is a 2D platformer built with Python (Pygame). Instead of a traditional hero, the protagonist is an exhausted worker. The challenge isn't defeating monsters, but dodging social pressures (the boss, family demands, and distractions) to finally reach home, drop the briefcase, and find the only true sanctuary: a partner's embrace.

🇪🇸 Español
Homebound es un juego de plataformas en 2D desarrollado en Python (Pygame). A diferencia de los héroes tradicionales, el protagonista es un trabajador agotado. El desafío no es derrotar monstruos, sino esquivar las presiones sociales (el jefe, las demandas familiares y las distracciones) para finalmente llegar a casa, soltar el maletín y encontrar el único refugio: el abrazo de su esposa.


🛠️ Tecnologias / Tech Stack / Tecnologías:
Language: Python 3.x
Library: Pygame
Concepts: Game Loop, Collision Detection (AABB), Gravity Physics, Sprite Management.


🚀 Como rodar / How to run / Cómo ejecutar:

1 - Clone o repositório / Clone the repo:
git clone https://github.com/seu-usuario/homebound.git

2 - Instale o Pygame / Install Pygame:
pip install pygame

3 - Execute o jogo / Run the game:
python main.py


🎮 Controles / Controls:
Left / Right Arrows: Mover / Move / Moverse

Space: Pular / Jump / Saltar
